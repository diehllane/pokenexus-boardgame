Google Sheets Setup Guide

Overview

The board game reads from three tabs in your Google Sheet: Players, BoardConfig, and FlavorText. The sheet must be published publicly (File → Share → Publish to web → CSV). The web app reads it on every page refresh, so any changes you make in the sheet will appear after refresh.

Step 1 — Publish your Sheet

  • Open your Google Sheet
  • Go to File → Share → Publish to web
  • Set format to CSV, publish the entire document
  • Copy the Sheet ID from the URL: docs.google.com/spreadsheets/d/SHEET_ID/edit
  • Paste the Sheet ID into the connection box at the top of this page

Step 2 — Players tab (Sheet name: Players)

Row 1 is the header. Add one player per row below. The app auto-detects however many rows exist — no player limit.

Gyms are tracked for badge counting — the player with the most gym landings wins the badge prize.

Step 5 — Connect and refresh

  • Paste your Sheet ID into the box at the top of the page and click Connect Sheet
  • The board reloads all config (squares, players, flavor text) on every page refresh
  • To move a bonus/penalty square: update its row in BoardConfig, then refresh the web app
  • To add players: add rows to the Players tab — no limit on player count
  • Player positions are read from Column E (TotalPosition) in the Players tab
